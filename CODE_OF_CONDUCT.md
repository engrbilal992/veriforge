# Code of Conduct

## Our Pledge

We are committed to making VeriForge an open, welcoming, and respectful community.

## Expected Behavior

Contributors are expected to:

- Be respectful and constructive
- Accept feedback gracefully
- Focus on improving the project
- Help others learn and contribute

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or discrimination
- Personal attacks
- Trolling or disruptive behavior
- Publishing others' private information

## Enforcement

Project maintainers may remove contributions or restrict participation for behavior that violates this code.

Thank you for helping make VeriForge a better project.
